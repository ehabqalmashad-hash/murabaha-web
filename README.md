# شمس المصارف - حاسبة المرابحة

تطبيق ويب احترافي لحساب المرابحة المالية للموظفين العراقيين.

## الميزات

- ✅ حاسبة مرابحة متقدمة
- ✅ دعم الوزارات المختلفة (داخلية + تربية، بقية الوزارات)
- ✅ سجل الحسابات مع الحفظ التلقائي
- ✅ الوضع الليلي والفاتح
- ✅ الإشعارات المخصصة
- ✅ مشاركة وتحميل النتائج
- ✅ واجهة احترافية وسهلة الاستخدام

## التكنولوجيا

- React 19
- TypeScript
- Vite
- CSS3

## التثبيت والتشغيل

```bash
# تثبيت المكتبات
npm install

# تشغيل الموقع في وضع التطوير
npm run dev

# بناء الموقع للإنتاج
npm run build

# عرض الموقع المبني
npm run preview
```

## الاستخدام

1. أدخل راتبك الشهري
2. اختر جهتك (داخلية أو أخرى)
3. أدخل مبلغ التمويل المطلوب
4. حدد سعر الفائدة والمدة
5. اضغط على "احسب المرابحة"
6. اعرض النتائج وشاركها أو حملها

## الحد الأقصى للتمويل

- **وزارة الداخلية + التربية**: 12 راتب شهري
- **بقية الوزارات**: 10 رواتب شهرية
